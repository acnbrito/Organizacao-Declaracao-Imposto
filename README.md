# Ferramenta de Organização para Declaração de Imposto de Renda
Descrição
Este repositório contém a entrega do projeto desenvolvido no curso Excel com IA da DIO.Me.
O objetivo é aplicar conceitos de Excel na criação de uma ferramenta prática que auxilie na organização e consolidação de informações essenciais para a declaração de imposto de renda. A solução reúne dados de forma validada, com menus de navegação, verificações automáticas e funcionalidades extras, como links rápidos, garantindo eficiência e facilidade de uso.

Objetivo do Projeto
1. Construir uma ferramenta que permita ao usuário:
2. Centralizar e organizar informações necessárias para a declaração de imposto de renda;
3. Validar automaticamente os dados inseridos, garantindo consistência;
4. Navegar de forma simples e intuitiva por meio de menus;
5. Acessar rapidamente funcionalidades extras que tornam o processo mais ágil e confiável.
